## 형변환
- 함수
```sql
CAST (값 as 데이터_형식 [(길이)])
CONVERT (값, 데이터 형식 [(길이)])
```
- 명시적인 변환
```sql
SELECT convert( AVG(price), signed integer) as '평균가격' from buy;
```
```sql
SELECT num, CONCAT(cast(price as char), 'x', cast(amount as char), '=') '가격 x 수량', price * amount '구매액'
from buy;
```
- concat() : 문자 연결


- 암시적인 변환 : 함수 사용 X.
```sql
select concat('100', '200'); -- 100200
select 100 + '200'; -- 300
```
